# Personal Budget PWA

Offline-first personal budget manager built with React.

## Tech Stack
- React 18 + TypeScript
- Vite
- TailwindCSS
- IndexedDB (Dexie)
- Zustand
- Zod

## Key Features
- Multi-account support
- Atomic balance updates
- Mobile-first quick add flow
- Offline-first persistence

## Status
MVP in progress.  
Reconciliation and AI-assisted categorization coming next.
